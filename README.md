# Finance Tracker — бэкенд

## Запуск

```bash
npm install
npm run start:dev
```

После запуска в консоли появятся две ссылки:

```
Server is running on http://localhost:4000
Swagger docs on    http://localhost:4000/docs
```

По второй ссылке — Swagger: там перечислены все эндпоинты, и запросы можно
отправлять прямо из браузера.

## Команды

| Команда | Что делает |
| --- | --- |
| `npm run start:dev` | Запуск в режиме разработки с автоперезагрузкой |
| `npm run build` | Сборка проекта в папку `dist` |
| `npm run start:prod` | Запуск собранного проекта |

## Настройки

Переменные лежат в `.env`. Для локальной работы менять их не обязательно.

| Переменная | Назначение |
| --- | --- |
| `PORT` | Порт бэкенда. По умолчанию `4000`, чтобы не конфликтовать с React |
| `CLIENT_URL` | Адреса фронтенда для CORS, можно несколько через запятую |
| `JWT_ACCESS_SECRET` | Секрет для access-токена |
| `JWT_REFRESH_SECRET` | Секрет для refresh-токена |
